## Comment ajouter un nouveau prospect

1. Créer un nouveau dossier : `nom-prospect/`
2. Y mettre un fichier `index.html`
3. Commiter
4. Attendre ~1 min le déploiement GitHub Pages
5. Le site sera disponible sur : `https://tenzintsultrim31.github.io/nom-prospect/`
